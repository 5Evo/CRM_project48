
Информация о госзакупках, размещенная на публичном FTP-сервере ООС. 


Адрес публичного FTP-сервера ООС:  ftp://free:free@zakupki.gov.ru.

На публичном FTP-сервере ООС расположены следующие каталоги:

1. 94fz          каталог, содержащий данные публичных выгрузок в соответствии с 94ФЗ (остальные каталоги - 44ФЗ),
2. fcs_nsi       каталог, содержащий данные нормативно-справочной информации,
3. fcs_regions   каталог, содержащий данные полной региональной выгрузки опубликованной на ООС информации,
4. fcs_banks     каталог, содержащий данные выгрузки опубликованной на ООС информации по банковским гарантиям,
5. fcs_rules     каталог, содержащий данные выгрузки по правилам (в настоящее время не реализована).
